
#+TITLE: AXI master

* 端子表

信号名 in/out

| 信号名    | in/out    | ビット幅           | 概略説明                                         |
|-----------+-----------+--------------------+--------------------------------------------------|
| axi_if    | interface | -                  | AXI インタフェース（AR/AW/W/R/B チャネル）       |
|-----------+-----------+--------------------+--------------------------------------------------|
| W_REQ_I   | input     | 1                  | 書き込み要求。1でAXI Writeトランザクションを開始 |
| W_ADDR_I  | input     | [P_A_BITWIDTH-1:0] | 書き込み先アドレス                               |
| W_LEN_I   | input     | [7:0]              | バースト長（AWLENに対応）                        |
| W_SIZE_I  | input     | [2:0]              | 転送サイズ（AWSIZEに対応）                       |
| W_DATA_I  | input     | [P_D_BITWIDTH-1:0] | 書き込みデータ                                   |
| W_VALID_I | input     | 1                  | 書き込みデータ有効（1beat単位）                  |
| W_READY_O | output    | 1                  | マスターが次データを受け取れるとき1              |
| W_LAST_I  | input     | 1                  | 最後のデータビートで1（WLASTに対応）             |
| W_DONE_O  | output    | 1                  | 書き込みトランザクション完了通知                 |
| W_RESP_O  | output    | [1:0]              | 書き込み応答コード（BRESP値）                    |
|-----------+-----------+--------------------+--------------------------------------------------|
| R_REQ_I   | input     | 1                  | 読み出し要求。1でAXI Readトランザクションを開始  |
| R_ADDR_I  | input     | [P_A_BITWIDTH-1:0] | 読み出しアドレス                                 |
| R_LEN_I   | input     | [7:0]              | バースト長（ARLENに対応）                        |
| R_SIZE_I  | input     | [2:0]              | 転送サイズ（ARSIZEに対応）                       |
| R_DATA_O  | output    | [P_D_BITWIDTH-1:0] | 読み出しデータ                                   |
| R_VALID_O | output    | 1                  | 読み出しデータ有効（RVALIDに対応）               |
| R_STRB_O  | output    | [P_S_BITWIDTH-1:0] | 読み出しデータストローブ                         |
| R_READY_I | input     | 1                  | 親モジュールがデータを受け取れるとき1            |
| R_LAST_O  | output    | 1                  | 最後のデータビートで1（RLASTに対応）             |
| R_DONE_O  | output    | 1                  | 読み出しトランザクション完了通知                 |
| R_RESP_O  | output    | [1:0]              | 読み出し応答コード（RRESP値）                    |
|-----------+-----------+--------------------+--------------------------------------------------|



* 注意
アンアラインドアクセス非対応
w_sizeが P_D_BITWIDTH以下のとき、右詰めで信号を入力すること
r_sizeが P_D_BITWIDTH以下のとき、右詰めで信号が出力されます
